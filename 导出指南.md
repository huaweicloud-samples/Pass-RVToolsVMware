# Skill开发流程图 - 导出为PPT格式指南

## 方法一：浏览器打印为PDF → PowerPoint（推荐）

### 步骤1：导出为PDF

1. 在浏览器中打开 `skill-flow-ppt.html`
2. 点击页面右上角的「打印/导出PDF」按钮，或按 `Ctrl+P` (Windows) / `Cmd+P` (Mac)
3. 在打印对话框中：
   - 选择「目标打印机」为「另存为PDF」或「Microsoft Print to PDF」
   - 选择「纸张大小」为「A4横向」或「16:9」
   - 确保勾选「背景图形」
4. 点击「保存」

### 步骤2：在PowerPoint中打开PDF

1. 打开PowerPoint
2. 点击「文件」→「打开」
3. 选择刚才保存的PDF文件
4. PowerPoint会自动将PDF转换为可编辑的幻灯片
5. 保存为 .pptx 格式

## 方法二：使用第三方工具Pandoc

如果你安装了Pandoc，可以使用命令行转换：

```bash
# 需要先安装wkhtmltopdf和pandoc
pandoc skill-flow-ppt.html -o skill-flow.pptx
```

## 方法三：截图/录屏方式（最简单）

1. 在浏览器中打开页面
2. 使用Windows自带的「截图工具」或「Win+Shift+S」
3. 选择全屏截图
4. 在PowerPoint中粘贴图片

## 技巧说明

### 打印设置建议

- 纸张大小：16:9（13.333 × 7.5 英寸）或 A4 横向
- 边距：无
- 布局：横向
- 勾选「背景图形」

### PowerPoint导入技巧

- 导入PDF后，可能需要调整文本框和图片位置
- PowerPoint会将PDF的每页转换为一张幻灯片
- 可以使用「格式刷」快速统一格式
