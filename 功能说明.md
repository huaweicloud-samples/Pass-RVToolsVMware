# 华为云虚机规格映射和汇总对比功能说明

## 功能概述

已成功在 `index.html` 中添加华为云虚机规格映射和汇总对比功能。

## 新增功能

### 1. 华为云虚机规格映射

#### 华为云ECS规格数据
- 创建了包含48种常用ECS规格的数据库
- 涵盖多种实例族：c7, s7, x1, x2, c6, s6, m7, g6
- 包含完整的规格信息：vCPU、内存、最大网卡数、X系列标识、价格

#### ECS规格匹配算法
实现了智能匹配算法，匹配规则：
1. vCPU >= 源端vCPU
2. 虚机CPU内存比接近源端虚机CPU内存比
3. 虚机最大网卡数量 >= 源端网卡数量
4. 优先X系列高性能实例
5. 选择价格最低的规格

#### EVS存储选择
- 数据库VM：选择通用型SSD V2
- 其他VM：选择高IO存储

#### 价格计算
- 支持按Region和货币类型自动计算价格
- 包含Region价格因子调整
- 计算月度费用和年度费用

### 2. 汇总对比

#### 源端资源统计
- VMware VM总量
- vCPU总量
- 内存总量
- 硬盘总量

#### 华为云资源映射
- VM总量
- vCPU总量
- 内存总量
- 硬盘总量
- 月度总费用
- 年度总费用

#### 对比表格
- 展示源端vs华为云的资源差异
- 颜色标识差异状态（绿色正常，黄色警告，红色不足）
- 提供详细说明和建议

### 3. 数据库VM识别

智能识别可能的数据库VM：
- OS名称包含数据库关键词：SQL, Oracle, MySQL, PostgreSQL, MongoDB, DB2, Redis等
- CPU >= 4 且 Memory >= 8GB 的Windows Server

### 4. 华为云价格数据

#### ECS实例价格
- 基于华北-北京四(cn-north-4)的基准价格
- 提供CNY和USD两种货币支持

#### EVS存储价格
- 高IO: ¥0.35/GB/月
- 通用型SSD V2: ¥0.48/GB/月
- ultra: ¥0.55/GB/月

#### Region价格因子
- 不同Region有不同价格系数
- 如：cn-hk-1 (香港) 为1.3倍，cn-southwest-2 (贵阳) 为0.9倍

## 实现的技术细节

### 核心函数
1. `isDatabaseVM()` - 数据库VM识别
2. `matchECSInstance()` - ECS规格匹配
3. `getStorageType()` - 存储类型选择
4. `getEVSPrice()` - EVS价格获取
5. `calculateMonthlyCost()` - 月度费用计算
6. `categorizeVM()` - VM分类
7. `renderHuaweiMapping()` - 渲染华为云映射表
8. `renderSummaryComparison()` - 渲染汇总对比

### 数据结构
```javascript
// ECS规格示例

  memory: 4,  // GiB

  isX: false,

  priceUSD: 26
```

### 集成点
- 修改了 `processData()` 函数签名，接收region参数
- 在数据处理完成后自动调用映射和汇总渲染函数
- 支持Region和货币类型动态切换

## 使用说明

1. 选择华为云Region
2. 选择货币类型（CNY/USD）
3. 上传RVTools Excel文件
4. 系统自动进行虚机规格映射
5. 在"华为云虚机规格映射"section查看详细映射表
6. 在"资源汇总对比"section查看源端vs华为云对比

## 注意事项

- 价格数据为参考价，实际价格以华为云官网为准
- 映射结果仅供参考，建议根据实际业务需求调整
- 存储大小按实际使用量向上取整
- 支持的数据库关键词可按需扩展
